# EasyInput
**A simple tool that lets you simulate keyboard input.<br>**
该工具适用于一些无法粘贴而只能使用键盘输入的场景。<br>
食用方式：<br>
-打开软件后软件默认置顶 <br>
-复制你想要输入的东西之后点击软件文本框<br>
-点击开始模拟输入<br>
-在倒计时结束前选中你要输入的程序或网页文本框（输入完成前请不要点击鼠标）<br>
**_请勿用于违法用途！！！_**<br>
